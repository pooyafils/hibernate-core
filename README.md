# hibernate-core
this application demostrad the usage of hibernate core for CRUD opratiation and relation between two tables such as one to one, one to many and many to one
## pre-installation
you need to have java 8, maven, mysql already installed
## configuration
after you clone this repository you need to create tables from the file ``sqldata.sql`` that you can find it in resoure folder.after that you should configure database info on the file named ``hibernate.cfg.xml``
